# 🎓 Cram Course Skill

[![CI](https://github.com/WXK2905821189/cram-course-skill-/actions/workflows/ci.yml/badge.svg)](https://github.com/WXK2905821189/cram-course-skill-/actions/workflows/ci.yml)
[![License: MIT](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

**把大学课程课件变成一套能直接复习的 Obsidian 笔记、题库和考前计划。**

期末快到了，PPT 还一堆没看？把课件丢给 Codex，让 `cram-course` 帮你整理成“先看什么、怎么背、刷什么题、哪里要回头补”的复习包。

适合会用 Codex 的大学生。安装一次，以后每门课都能直接开跑。

## ⚡ 安装

把这句话发给 Codex：

```text
请安装这个 skill：https://github.com/WXK2905821189/cram-course-skill-/tree/main/cram-course
```

安装完成后，重启 Codex。

## 🚀 使用

把课程 PPT、PDF、Word 讲义、课堂笔记或题库拖给 Codex，然后发送：

```text
阅读 cram-course。请根据我上传的课件生成期末速通资料。考试日期是 2026-06-28，每天能学 4 小时，输出为 Obsidian Markdown 笔记。
```

也可以把老师划的重点、考试范围、历年卷一起发过去，效果会更贴近你的课程。

## ✨ 它能帮你做什么

- 🧭 整理课程地图：先知道这门课到底在讲什么。
- 📝 生成章节笔记：把课件压成能复习的短笔记。
- 🧪 生成章节题库：每章配练习题、答案和解释。
- 🎯 生成专项训练：名词解释、公式计算、对比辨析、简答论述、综合应用。
- 🔁 安排复习节奏：按考试日期做间隔复习计划。
- 🧯 整理错题本：把“看着会，做就错”的地方揪出来。
- ⏱️ 生成模拟考试：考前用一套卷子压一遍。

## 📦 会生成哪些文件

文件名默认使用中文，并加上 `00`、`01`、`10` 这样的前缀，放进 Obsidian 后会自然排序。

| 文件 | 用途 |
|---|---|
| `00_课程导学.md` | 课程地图、重点区域、陌生度标记和导学问题 |
| `01_学习计划.md` | 按考试日期和每天可学习时间安排冲刺节奏 |
| `02_覆盖矩阵.md` | 对照 slide/page/section，检查有没有漏掉课件内容 |
| `10_第01章_<章节名>_01学习笔记.md` | 每章一份可复习的短笔记 |
| `10_第01章_<章节名>_02章节题库.md` | 每章一份练习题，含答案和解释 |
| `30_名词解释专项.md` | 适合背概念、准备名词解释 |
| `31_公式计算专项.md` | 公式、计算题、单位陷阱和变式题 |
| `32_对比辨析专项.md` | 容易混淆的概念、判断题和对照表 |
| `33_简答论述专项.md` | 简答题、论述题、答题框架 |
| `34_综合应用专项.md` | 跨章节大题和综合应用题 |
| `40_间隔复习计划.md` | 1、3、7、14 天复习安排和考前回看 |
| `41_错题本.md` | 按错误类型整理错题和复盘动作 |
| `50_模拟考试.md` | 考前模拟卷、评分点、参考答案和最后 24 小时清单 |

## 💬 可以直接复制的提示词

**三天速通：**

```text
阅读 cram-course。我还有 3 天考试，每天能学 5 小时。请根据我上传的课件生成一套优先保分的期末速通资料，输出为 Obsidian Markdown。
```

**老师划了重点：**

```text
阅读 cram-course。请重点参考我发的考试范围和老师强调内容，再结合课件生成复习资料。不要漏掉考试范围里的任何一项。
```

**只剩一天：**

```text
阅读 cram-course。我明天考试，请按紧急模式生成核心笔记、高频题、模拟卷和最后 24 小时复习清单。
```

## 📚 支持的材料

- `.pptx`
- `.docx`
- 带文本层的 `.pdf`
- 课堂笔记、考试范围、老师重点、题库、历年卷

旧版 `.ppt` 建议先转成 `.pptx`。扫描版 PDF 建议先 OCR，否则 AI 可能读不到里面的文字。

## 🧪 示例

- `examples/full-demo/`：一个迷你数据结构课程的完整输出示例
- `examples/sample_study_materials.json`：结构化输入示例
- `examples/示例_40_间隔复习计划.md`：间隔复习计划示例

## 🔒 使用提醒

- 不要把受版权保护的课程课件上传到公开仓库。
- 如果通过在线 AI 服务处理课件，课件内容可能会发送到对应服务商，请先确认学校或老师的要求。
- AI 生成的考点、题目和答案需要人工复核，尤其是公式、计算题和老师特别强调的内容。
