# Full Demo: Data Structures Mini Cram

This folder shows the shape of a tiny generated output package for a data structures course.

It is intentionally small and copyright-safe. Real outputs should include one notes file and one quiz file per chapter, all drill files, a review schedule, an error log, and a mock exam.

Included demo artifacts:

- `01_学习计划.md`
- `02_覆盖矩阵.md`
- `10_第01章_数据结构基础_01学习笔记.md`
- `10_第01章_数据结构基础_02章节题库.md`
