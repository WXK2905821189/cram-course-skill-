# 第 1 章：数据结构基础

## 栈

**一句话定义**：栈是只允许在一端插入和删除的线性表，特点是后进先出。

**例子**：浏览器后退记录；函数调用栈。

**自我解释**：为什么递归可以用栈模拟？

## 队列

**一句话定义**：队列是一端入队、另一端出队的线性表，特点是先进先出。

**例子**：排队买饭；任务调度队列。

## 时间复杂度

时间复杂度描述算法运行时间随输入规模增长的趋势。

| 复杂度 | 含义 |
|---|---|
| O(1) | 常数时间 |
| O(n) | 线性增长 |
| O(n log n) | 常见高效排序复杂度 |

## 一句话考点

1. 栈是后进先出。
2. 队列是先进先出。
3. 栈适合处理递归和回退。
4. 队列适合处理排队和调度。
5. 时间复杂度关注增长趋势，不关注机器速度。
