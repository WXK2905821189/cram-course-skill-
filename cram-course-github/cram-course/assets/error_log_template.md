## 错题分类标准

| 类型 | 标签 | 判断标准 | 应对策略 |
|------|------|---------|---------|
| 概念不清 | 🔴 | 选错选项、判断反了、简答完全偏离 | 重回笔记 + 做自我解释 + 找 2 个具体例子 |
| 记忆遗忘 | 🟡 | 记得学过但想不起来、公式记错 | 加入间隔重复队列 + 记忆术辅助 |
| 粗心失误 | 🟢 | 看错题干、单位漏换、计算粗心 | 标注陷阱类型 + 建立"常见坑"清单 |
| 应用困难 | 🔵 | 概念懂但不会做题、只会套公式不会变通 | 做变式题 3 道 + 合上笔记写解题思路 |

## 考前冲刺优先级

按 **错题类型** × **考试占比** 排序：

1. 🔴 概念不清 + 高频考点 → **最高优先级**
2. 🔵 应用困难 + 高频考点
3. 🟡 记忆遗忘 + 高频考点
4. 🔴 概念不清 + 中频考点
5. 🟢 粗心失误（全部检查一遍即可）
