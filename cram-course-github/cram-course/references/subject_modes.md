# Subject Modes

Use this to adapt notes, quizzes, drills, and mock exams to the course type. If the course blends multiple modes, combine the closest two.

## Mode Table

| Subject mode | Typical courses | Emphasize | Question mix |
|---|---|---|---|
| CS/math/engineering | data structures, calculus, physics, circuits | definitions, formulas, proofs, steps, complexity, edge cases | calculation, derivation, coding/pseudocode, variants |
| economics/management | microeconomics, accounting, marketing, operations | concepts, assumptions, models, comparisons, cases | definitions, comparison, calculation, case analysis |
| law/politics/humanities | law, politics, philosophy, literature | terms, schools, arguments, source texts, essay frames | definitions, short answer, essay, material analysis |
| medicine/biology | anatomy, physiology, biochemistry, pathology | mechanisms, classification, processes, symptoms, exceptions | flow charts, comparison, mechanism explanation, case diagnosis |
| language/exam prep | English, Japanese, writing, translation | vocabulary, grammar, patterns, examples, timed output | cloze, translation, rewriting, speaking/writing templates |

## Adaptation Rules

- For calculation-heavy courses, make `31_公式计算专项.md` deeper than `33_简答论述专项.md`.
- For essay-heavy courses, make `33_简答论述专项.md` and `50_模拟考试.md` include answer frameworks and scoring points.
- For concept-heavy courses, strengthen `32_对比辨析专项.md` and add more true/false traps.
- For memory-heavy courses, add mnemonic suggestions in notes and use shorter review intervals.
- For case-heavy courses, include scenario variations in chapter quizzes and comprehensive drills.
